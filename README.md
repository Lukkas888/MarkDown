# MarkDown
 Linguagem de marcação simplificada 

 ## **Conceito** 

  Criada para ser uma alternativa à linguagem HTML  

 ##  **Onde é Ultilizada** 

  1- *Readme.md* 
  2- *Issues* 
  3- *Pull-Requests*  

  ***

 ## **Alguns comandos** 

  (~~)  ~~Rasura~~ 
  (*)   *ITÁLICO* 
  (**)  **NEGRITO** 
  (#)   #Título de nível 01 
  (##)  ##Título de nível 02
